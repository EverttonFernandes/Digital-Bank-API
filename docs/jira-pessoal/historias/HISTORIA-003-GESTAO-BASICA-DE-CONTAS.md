# HISTORIA-003 — Gestão Básica de Contas

## Tipo

- funcional

## Objetivo da História

Disponibilizar contas pré-carregadas e permitir sua consulta pela API.

## Valor de Negócio

Esta história torna o domínio visível e cria a base operacional para a transferência entre contas.

## Critérios de Aceite

- deve existir seed inicial de contas
- deve ser possível listar contas
- deve ser possível buscar conta por identificador
- saldo e titular devem ser retornados corretamente
